如何搭建自己的博客

Over View

This is my first document this year and hope I can continue to record my work, life and thoughts. Today summary how to setup the this blog.

Life without recording can’t be review and without reviewing can’t be made progress.

  1. create the your blog site by github page

  2. Setup the local blog environment

  3. Use Mark edit to write your idea

  4. Sync new blog to github

Create the your blog site by github page

This is simple. Just create github account and create one respository with your name, like roczhang.github.io
Then, you can access it http://roczhang.github.io/
More details: https://guides.github.com/features/pages/

Setup local environment

Use hexo.
